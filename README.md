# Government Exam Mastery Tracker


A single-subject, layer-based 24-week preparation tracking system for government exam aspirants. 
It helps users log daily study, track weekly performance, and measure readiness score.

## Features

- Add Daily Study Logs
- Add Weekly Performance Logs
- Layer-Based Preparation System
- 24-Week Mastery Ladder
- Readiness Score Calculation
- Weekly Trend Visualization
- Analytics Section (Blocker Tracking)


  ## Tech Stack

- HTML
- CSS
- JavaScript


  ## How It Works

1. Select your subject.
2. Add daily logs (hours + outcome + proof).
3. Add weekly logs (effort, blocker, measurable outcome).
4. System calculates readiness score.
5. Track progress over 24 weeks.
